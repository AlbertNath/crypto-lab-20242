Instrucciones de Uso - KeyLogger
Requisitos Previos

Se necesitan instalar los siguientes modulos :
  - sudo su pip install keyboard yagmail keyrings.alt
Se instala el siguiente paquete :
  - sudo apt install sendemail


Para correr el programa se ejecuta -
  sudo su python KeyLogger.py


Configuración Adicional
Si planeas utilizar una cuenta de Google para enviar correos electrónicos, es necesario habilitar el acceso a aplicaciones menos seguras. Puedes hacerlo siguiendo estos pasos:

- Accede a tu cuenta de Google.
- Ve a la sección de Seguridad.
- Desplázate hacia abajo hasta encontrar la opción "Acceso de aplicaciones menos seguras".
- Habilita esta opción.


Notas Importantes
- Ten en cuenta que este programa captura las pulsaciones de teclas y puede ser utilizado únicamente con fines educativos o de investigación. El uso inadecuado o no autorizado de este programa puede violar las leyes de privacidad y seguridad.
